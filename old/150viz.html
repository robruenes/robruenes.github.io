Visualization

COMP150 - Special Topics in Visualization was taught my sophomore year at Tufts under Remco Chang, who also happens to be my advisor. All projects on this page were written in Processing.

Draw Box

The "draw box" project was an introduction to Processing and the tools available in its API. When clicked on, the box changes size and color.

Line and Bar Charts

The line and bar chart project involved dynamic data entry and resizing of the visualization based on the input data. Data is fed into the visualization via a .csv file, and the result is a system that can toggle back and forth between a line graph and bar chart displaying the data.

Theme River

An implementation of a visualization first described in this paper. As described by the abstract, the visualization is a "system that visualizes thematic variations over time across a collection of documents. The 'river' flows through time, changing width to depict changes in the thematic strength of documents temporally collocated. Themes or topics are represented as colored 'currents' flowing within the river that narrow or widen to indicate decreases or increases in the strength of a topic in associated documents at a specific point in time."

Squarified Treemaps

An implementation of a visualization first described in this paper. The visualization aims to extend the commonly used treemap method for displaying hierarchical information (like directory structures, for example). The standard treemap visualization typically results in areas of extremely thin rectangles. This extension aims to rectify the problem by generating the lay-out such that all the rectangles are approximate squares.

Force-Directed Node Link Diagram

An environment simulating a self-structuring graph. Individual nodes can be dragged around, exerting forces on the rest of the system to move along with it.
